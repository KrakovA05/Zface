## Дизайн

- [x] ✅ **Редизайн цветовой схемы — тёплая светлая пастель** — текущая тема холодная и тёмная (тёмно-синий + фиолетовый). Для аудитории с убитой менталкой не добавляет уюта. Нужна тёплая светлая палитра — молочный, бежевый, кремовый. Не клинически белый и не фальшиво-радостный, а как старая бумага — мягко и без агрессии.

  | Переменная | Сейчас | Новый цвет |
  |-----------|--------|------------|
  | `background` | `#0d0d14` | `#FAF7F2` — молочный с тёплым оттенком |
  | `card` | `#1a1a2e` | `#FFFFFF` или `#FFF9F2` |
  | `white` (основной текст) | `#ffffff` | `#2C2420` — тёплый тёмно-коричневый |
  | `muted` | `#6b6b8a` | `#8B7B6B` — тёплый коричнево-серый |
  | `border` | `#2a2a4a` | `#E8DFD0` — бежевая граница |
  | `accent` | `#7c3aed` | оставить или сдвинуть в `#9B6B9B` (пыльный сиреневый) |

  Захардкоженные места править вручную: `rgba(255,255,255,0.07)` в инпутах чатов, `#e74c3c` ошибки, `#2a2a4a` пузыри сообщений.

## Структура и смысл

- [x] ✅ **Комментарии к постам** — лайк не создаёт связь, комментарий создаёт. «Я через это тоже прошёл» — это и есть «найди своих». Без комментариев лента читается как доска объявлений, не как сообщество.

- [ ] 🔴 **Анонимный режим в комнатах** — войти без имени, просто присутствовать. «Сейчас здесь 5 человек. 3 из них просто сидят.» Уникальная механика которой нет в мессенджерах — прямо для аудитории которой плохо но говорить не готовы.

- [ ] 🟡 **Поиск по ярлыкам** — сейчас людей можно найти только по нику. Добавить фильтр в FriendsScreen: выбираешь ярлык → видишь людей с этим ярлыком. Запрос: `supabase.from('users').contains('labels', [label])`.

- [ ] 🟡 **Убрать BarScreen** — концептуально красивый, но по факту 4 обычных чата. Убрать экран, убрать из навигации, почистить `messages` от `level = 'bar_*'`.

- [ ] 🟢 **Профиль как история** — динамика уровня другого человека на его UserProfileScreen. «Шёл с red до yellow 3 недели» — это история и доверие. Данные уже есть в `test_results`.

## Удержание и рост

- [ ] 🔴 **Анонимная мысль дня + отложенные реакции** — главная петля возврата. Раз в день пишешь короткую анонимную мысль. Другие реагируют одним из трёх: «я понимаю» / «я тоже» / «держись». Завтра возвращаешься посмотреть реакции. Быть услышанным без последствий.

  - Одна мысль в день на пользователя
  - Реакции видны только автору
  - Мысли хранятся 7 дней
  - БД: `anonymous_thoughts` + `thought_reactions`

- [ ] 🔴 **Вопрос дня: показывать ответы других** — сейчас ответ видит только сам пользователь. Нужно показывать анонимные ответы других с того же уровня. Самый сильный механик «ты не один» — каждый день.

- [ ] 🔴 **Онбординг-момент после регистрации** — после теста показать: сколько человек сегодня с таким же уровнем + один анонимный ответ на вопрос дня от кого-то с тем же уровнем. Один экран — один момент «я не один».

- [ ] 🟡 **Одно слово дня** — каждый день новое слово-состояние на главном экране: «пустота», «тревога», «злость», «усталость». Одно касание — «да, это про меня». Видишь счётчик: «ещё 147 человек так же». 2 секунды, ежедневный ритуал. Слова в `constants.js` с привязкой к дате.

- [ ] 🟡 **Счётчики онлайна везде** — «сейчас в красной комнате 8 человек». Без этого приложение кажется пустым даже когда оно не пустое. Считается по `last_seen` который уже есть в БД.

- [ ] 🟡 **Динамика сообщества на главном экране** — «сегодня 83 человека с твоим уровнем заходили». Живое ощущение что ты не один. Считается по `last_seen` и `test_results` — новых таблиц не нужно.

- [ ] 🟡 **Проактивные пуши — мини-квесты** — приглашения что-то сделать, не просто уведомления. «Выдели 4 минуты. Зайди подышать» → BreathingScreen. «Закинь удочку» → FishingScreen. Тон без давления. Утро/обед/вечер, ночью не слать. Привязка к уровню: на `red` — мягче.

- [ ] 🟢 **Переосмыслить шер/инвайт** — шерить конкретный пост с подписью «узнаёшь себя?» вместо просто ссылки на приложение. Вирусно и не требует объяснений.

- [ ] 🟢 **Раздел с психологическими материалами (ResourcesScreen)** — список тем → статьи, подкасты, видео-разборы.

  Темы: тревога / апатия и депрессия / выгорание / отношения и одиночество / самооценка.

  Контент в `constants.js`, открытие через `Linking.openURL()`. Точка входа — модуль на HomeScreen.

- [ ] 🔴 **Политика конфиденциальности** — обязательна для App Store и Google Play. Простая страница или ссылка на внешний URL.
