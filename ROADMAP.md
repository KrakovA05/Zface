## Дизайн

- [x] ✅ **Редизайн цветовой схемы — тёплая светлая пастель** — текущая тема холодная и тёмная (тёмно-синий + фиолетовый). Для аудитории с убитой менталкой не добавляет уюта. Нужна тёплая светлая палитра — молочный, бежевый, кремовый. Не клинически белый и не фальшиво-радостный, а как старая бумага — мягко и без агрессии.

  | Переменная | Сейчас | Новый цвет |
  |-----------|--------|------------|
  | `background` | `#0d0d14` | `#FAF7F2` — молочный с тёплым оттенком |
  | `card` | `#1a1a2e` | `#FFFFFF` или `#FFF9F2` |
  | `white` (основной текст) | `#ffffff` | `#2C2420` — тёплый тёмно-коричневый |
  | `muted` | `#6b6b8a` | `#8B7B6B` — тёплый коричнево-серый |
  | `border` | `#2a2a4a` | `#E8DFD0` — бежевая граница |
  | `accent` | `#7c3aed` | оставить или сдвинуть в `#9B6B9B` (пыльный сиреневый) |

  Захардкоженные места править вручную: `rgba(255,255,255,0.07)` в инпутах чатов, `#e74c3c` ошибки, `#2a2a4a` пузыри сообщений.

## Структура и смысл

- [x] ✅ **Комментарии к постам** — лайк не создаёт связь, комментарий создаёт. «Я через это тоже прошёл» — это и есть «найди своих». Без комментариев лента читается как доска объявлений, не как сообщество.

- [x] ✅ **Анонимный режим в комнатах** — войти без имени, просто присутствовать. «Сейчас здесь 5 человек. 3 из них просто сидят.» Уникальная механика которой нет в мессенджерах — прямо для аудитории которой плохо но говорить не готовы.

- [x] ✅ **Поиск по ярлыкам** — уже реализован: chips в поиске FriendsScreen, `.overlaps('labels', selectedLabels)` в запросе.

- [x] ✅ **Убрать BarScreen** — экран удалён из навигации и HomeScreen, сообщения очищены из БД, бар убран из MessagesScreen и refreshBadges.

- [x] ✅ **Профиль как история** — динамика уровня другого человека на его UserProfileScreen. Свитч «Показывать мою динамику» в ProfileScreen (по умолчанию выключен). Только если включён — другие видят историю тестов.

## Удержание и рост

- [x] ✅ **Анонимная мысль дня + отложенные реакции** — главная петля возврата. Раз в день пишешь короткую анонимную мысль. Другие реагируют одним из трёх: «я понимаю» / «я тоже» / «держись». Завтра возвращаешься посмотреть реакции. Быть услышанным без последствий.

- [x] ✅ **Вопрос дня: показывать ответы других** — анонимные ответы других пользователей за сегодня появляются после того как ответил сам. Фиксы: локальная дата вместо UTC, убран фильтр по уровню.

- [x] ✅ **Онбординг-момент после регистрации** — после первого теста показывает: сколько человек с таким же уровнем + один анонимный ответ на вопрос дня. Экран OnboardingMomentScreen → Recommendations.

- [x] ✅ **Одно слово дня** — карточка на HomeScreen, 25 слов-состояний в `constants.js`. Тап «да, это про меня» → счётчик «ты и ещё N человек чувствуют то же». Таблица `daily_word_taps` в БД.

- [x] ✅ **Счётчики онлайна везде** — в лобби комнат и MessagesScreen показывают пользователей с `last_seen > 10 минут назад`. `last_seen` обновляется каждые 2 минуты.

- [x] ✅ **Динамика сообщества на главном экране** — «сегодня 83 человека с твоим уровнем заходили». Живое ощущение что ты не один. Считается по `last_seen` и `test_results` — новых таблиц не нужно.

- [x] ✅ **Проактивные пуши — мини-квесты** — приглашения что-то сделать, не просто уведомления. «Выдели 4 минуты. Зайди подышать» → BreathingScreen. «Закинь удочку» → FishingScreen. Тон без давления. Утро/обед/вечер, ночью не слать. Привязка к уровню: на `red` — мягче.

- [ ] 🟢 **Прокачка рыбалки** — расширить FishingScreen: разные виды рыб с разной редкостью (зависит от уровня пользователя или времени суток), коллекция пойманных рыб, возможные достижения за уловы. Сделать из простой мини-игры полноценную медитативную активность с прогрессией.

- [ ] 🟢 **Переосмыслить шер/инвайт** — шерить конкретный пост с подписью «узнаёшь себя?» вместо просто ссылки на приложение. Вирусно и не требует объяснений.

- [ ] 🟢 **Раздел с психологическими материалами (ResourcesScreen)** — список тем → статьи, подкасты, видео-разборы.

  Темы: тревога / апатия и депрессия / выгорание / отношения и одиночество / самооценка.

  Контент в `constants.js`, открытие через `Linking.openURL()`. Точка входа — модуль на HomeScreen.

## Безопасность и публикация

- [x] ✅ **Поддержка в кризисе** — обязательно для App Store и Google Play (mental health). Кнопка «если совсем плохо» → телефон доверия (8-800-2000-122, бесплатно). `Linking.openURL('tel:88002000122')`.

- [ ] 🔴 **Политика конфиденциальности** — обязательна для App Store и Google Play. Простая страница или ссылка на внешний URL.

- [ ] 🔴 **Firebase FCM для Android** — для production push-уведомлений на Android нужны FCM-credentials в EAS. Бесплатно. Нужно перед публикацией в Google Play.

- [x] ✅ **Подтверждение email при регистрации** — код реализован и проверен (`EmailConfirmScreen`, флаг `EMAIL_CONFIRM_ENABLED` в `config.js`). Сейчас включено (`true`). Перед релизом: настроить deep link чтобы письмо открывало приложение, а не браузер.
